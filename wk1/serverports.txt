• What is the definition of server port?
- Server ports are the endpoints(logical) of a network conection which is used for information exchange between a client and a server.
These ports have a unique number so that they can be identified separately.

• What is server port range?
-The range of the server ports are from 0 to 65535. However, only 0 to 1023 port numbers are reserved (for priviledged services) and are designated as well-known ports.
